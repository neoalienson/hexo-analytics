# hexo-analytics
A Hexo plugin to enable Google Analytics, Tag Manager and Microsoft Clarity

## Configurations

### Enable the plugin

To enable the plugin add this to your config.yml:
```
analytics:
    google_analytics:
        enable: true
    google_tag_manager:
        enable: true
    microsoft_clarity:
        enable: true
        project_id: <your project id under Setting, Overview>
```
